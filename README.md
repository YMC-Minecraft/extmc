# extmc

Minecraft plugin framework based on stdout and rcon.

## Features

* Plugin system.
* Multithreaded.
* Almost all settings can be reloaded at runtime.

## Requirements

* POSIX.1-2008 compliant system (BSD support is on the way)
* C11 compiler
* GNU Make
